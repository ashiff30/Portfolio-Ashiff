INTRODUCTION

Hi, I'm Ashiff — an enthusiastic Technical Support and Web Design professional with a foot in both network operations. At Tech Mahindra, I work with ArcGIS, Salesforce, GeoHUB, and Orion to diagnose network enquiries and keep service delivery running smoothly for a UK telecom infrastructure client.

Alongside that, I build responsive, user-friendly interfaces with HTML, CSS, Bootstrap, and React. The two disciplines feed each other: the same habit of tracing a problem to its root in a ticket queue is what I use to debug a layout or a broken component.

This portfolio holds a handful of projects — from responsive layouts to data analysis — that reflect that mix of technical precision and creative problem-solving.
